## How To Use

```bash
# Clone this repository
$ git clone https://github.com/charlesmarafao/hotel-app.git

# Go into the repository
$ cd hotel-app

# install dependencies
$ yarn

# You can see the app running at: http://localhost:3000
$ yarn dev
```
